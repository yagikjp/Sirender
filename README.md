# Sirender
"Sirender" is an emergency vehicle siren detecting software for IoT device e.g. M5StickC Plus.

サイレンダーは緊急車両のサイレンを検知するソフトウェアで、M5StickC PlusのようなIoTデバイス向けに作りました。
車載してサイレンが聞こえたときに表示を出して知らせるものです。

検知精度はあまり良くありません。
改良するかも知れないし、しないかも知れないし、M5StickC Plusで音を取得してFFTするサンプルプログラム的にどうぞ。
